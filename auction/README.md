# auction

## Quick Start
```
npm i			# Install dependencies.
npm start		# Start server, compile/bundle all sources, and rebuild on changes.
```

## All Scripts
```
npm run build   # Compile and bundle all sources.
npm run watch	# Compile and bundle all sources and rebuild on changes.
npm run server	# Start HTTP server on localhost:8080.
npm start		# Start server, compile/bundle all sources, and rebuild on changes.
```
